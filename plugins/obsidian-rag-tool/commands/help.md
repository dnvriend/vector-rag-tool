Display help information for obsidian-rag-tool CLI commands.

## Usage

```bash
# Show general help
obsidian-rag-tool --help

# Show command-specific help
obsidian-rag-tool COMMAND --help

# Show version
obsidian-rag-tool --version
```

## Arguments

- `COMMAND` (optional): Specific command to get help for
- `--help` / `-h`: Show help information
- `--version` / `-v`: Show version information

## Examples

```bash
# General help
obsidian-rag-tool --help

# Command help
obsidian-rag-tool search --help

# Version information
obsidian-rag-tool --version
```

## Output

Displays usage information, available commands, and options.
